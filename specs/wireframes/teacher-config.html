Teacher Classroom Configuration

Screen: teacher-config | Persona: PER-002 (Ms. Torres the Teacher) | Journey: JRN-003 (classroom setup)

Information Hierarchy

1
AI Behavior Configuration (top section) — Teaching mode selector (Socratic/Direct/Balanced), subject focus, grade band, and tone. This is the most important config — it determines how the AI interacts with students.
2
App Enablement (center section) — Toggle grid of district-approved apps. Teacher enables/disables per classroom. Each app shows name, description, and a toggle.
3
Classroom Details (top area) — Period, subject, grade, student count. Read from SIS integration. Editable classroom name.
4
Consent Status (bottom section) — Table showing which students have parental consent, which are pending, which were denied. Resend consent link action.

Focal Point

FOCAL AI Behavior Configuration is the focal point because it answers the teacher's key question: "How will the AI behave with my students?" The teaching mode selector (Socratic vs. Direct) is the single most important choice.

Above-the-fold: Classroom details + AI behavior configuration visible without scrolling. App enablement and consent status below the fold.

Banned Dominance Patterns

  • No advanced/technical settings visible by default — show "Advanced" expandable for power users
  • No JSON or code editing for AI configuration — use labeled selectors and toggles
  • Template presets must be prominent for first-time setup ("Science — Socratic", "Math — Direct")
  • Consent status must not dominate the page — it's informational, not the primary config task

Interactive States

CLICK Teaching mode selector: Three-option segmented control (Socratic / Balanced / Direct). Changing mode shows a preview of how the AI would respond to "What is photosynthesis?" in that mode.

CLICK Template presets: Preset buttons ("6th Grade Science", "AP Physics", "Elementary Math") auto-fill all config fields. Teacher can customize after applying a preset.

TOGGLE App toggles: Enable/disable apps per classroom. Only district-approved apps appear. Disabled apps show as grayed out toggles.

CLICK Save button: Saves configuration. Shows success toast: "Configuration saved. Changes will apply to new sessions." Active sessions continue with old config.

CLICK Resend consent link: Per-student action to resend the parental consent email. Shows confirmation: "Consent link sent to parent/guardian."

INPUT Subject boundaries: Multi-select tags for allowed subjects. AI will redirect students who ask about off-topic subjects: "That's a great question, but this chat is focused on Science."

API Integration Annotations

API Load config: GET /api/v1/classrooms/{classroom_id}/config — returns current teaching mode, subject focus, grade band, tone, enabled apps, consent status per student.

API Save config: PUT /api/v1/classrooms/{classroom_id}/config — body includes all config fields. Validated server-side (e.g., cannot enable a suspended app).

API Available apps: GET /api/v1/apps?district_id={id}&status=approved — list of apps the teacher can enable for this classroom.

API Consent status: GET /api/v1/classrooms/{classroom_id}/consent-status — per-student consent state (approved/pending/denied).

API Resend consent: POST /api/v1/students/{student_id}/resend-consent — triggers consent email to parent/guardian.

API Config preview: POST /api/v1/classrooms/{classroom_id}/config/preview — body: { "mode": "socratic", "prompt": "What is photosynthesis?" }. Returns sample AI response for the selected mode.

Responsive Behavior

MOBILE 375px Config sections stack vertically. App toggles become a single-column list. Consent table switches to card view. Teachers primarily configure on desktop/laptop before class.

DESKTOP 1280px Two-column layout: AI config and app toggles side-by-side. Consent status as a table below. Max-width 1000px centered.

Persona Fit

PERSONA Teachers configure classrooms before the school year starts and occasionally adjust mid-semester. The template presets are essential for onboarding — a teacher should be able to set up a classroom in under 2 minutes by selecting a preset and enabling apps. The Socratic/Direct mode selector is the product's key differentiator from generic AI chatbots. The AI behavior preview builds teacher confidence that the AI will behave as expected. Consent status tracking reduces the "whose parent hasn't signed?" burden that falls on teachers.
